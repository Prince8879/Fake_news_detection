# Fake News Detection 📰🔍

## 📌 Project Overview

Fake News Detection is a Machine Learning project that classifies news articles as **Real** or **Fake** using Natural Language Processing (NLP) techniques.
The goal of this project is to help identify misleading or false news content automatically.

---

## 🚀 Features

* Detects whether a news article is **Real or Fake**
* Uses **Machine Learning algorithms**
* Text preprocessing and feature extraction
* Trained on a news dataset
* Implemented using Python

---

## 🛠️ Technologies Used

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Natural Language Processing (NLP)
* Jupyter Notebook / Google Colab

---

```
Fake_news_detection
```

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/Prince8879/Fake_news_detection.git
```

2. Navigate to the project folder:

```
cd Fake_news_detection
```

3. Install required libraries:

```
pip install pandas numpy scikit-learn
```

---

## ▶️ How to Run

1. Open the notebook in **Google Colab** or **Jupyter Notebook**
2. Run all cells step by step
3. Enter a news text to check if it is **Fake or Real**

---

## 📊 Machine Learning Model

The project uses a **text classification model** trained with:

* TF-IDF Vectorization
* Logistic Regression / Naive Bayes (depending on your model)
